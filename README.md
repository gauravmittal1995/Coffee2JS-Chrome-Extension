Coffee2JS
=========



1 DESCRIPTION
-------------

**Coffee2JS** is a chrome extension which allows you to test your coffeeScript by compiling it to javaScript.

2 INSTALL
---------

Simply Clone the Repository and load it as an external extension on your chrome browser.
